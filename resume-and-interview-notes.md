# Resume and Interview Notes

## Resume Project 1

**Smart Ticket API | Ruby, Sinatra, SQLite, RSpec, Docker**

- Built a Ruby-based REST API for customer support ticket management with ticket creation, listing, status updates, and automatic priority scoring.
- Implemented clean service objects, request validation, SQLite persistence, and JSON API responses using Sinatra and Sequel.
- Added RSpec and Rack::Test coverage for health checks, ticket creation, filtering, and status transitions.
- Dockerized the application with Docker Compose for repeatable local setup and simple deployment.

## Resume Project 2

**Container Stack Monitor | Docker, Docker Compose, Ruby, Redis, Nginx**

- Created a multi-container monitoring stack using Docker Compose with Ruby API, Redis storage, and Nginx reverse proxy.
- Configured container health checks, environment-based settings, service networking, and operational `/health`, `/services`, and `/metrics` endpoints.
- Implemented lightweight service status tracking and Prometheus-style metrics output for DevOps monitoring workflows.
- Documented build, run, test, and troubleshooting commands for recruiter-friendly GitHub presentation.

It shows that I understand Docker Compose, service networking, health checks, and operational endpoints.
